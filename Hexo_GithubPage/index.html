Set Up and Deploy Hexo Blog to Github Pages

Github Pages

Create a new repository named username.github.io, where username is your username (or organization name) on GitHub.

Hexo

  1. install Node.js
  2. install Git (Windows)
  3. install Hexo npm install -g hexo-cli
  4. create a local blog
    • create a blog folder
    • in the blog folder, run hexo init to initialize the hexo blog
    • run npm install to install all the dependencies
    • hexo g will generate the static webpage
    • hexo s will run the server in http://localhost:4000/
  5. create a new post hexo new 'My First Post'

Deploy the Hexo blog to Github Pages

  1. install hexo-deployer-git npm install hexo-deployer-git --save
  2. Then modify the property deploy of the file _config.yml
  3. push the hexo blog to github pages repository: hexo d

Change Theme

The theme I used for my personal homepage is minos.

Install theme:

Then modify the property theme of the file hexo/_config.yml to theme: minos.

The detail installation and moditication guide can be found here.
