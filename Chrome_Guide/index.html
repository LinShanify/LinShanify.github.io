List Some Tips and Tricks for Chrome

Quick Switch Different Search Enging in Address Bar

  1. Enter Chrome Setting: chrome://settings/

  2. Edit Mange Search Engines

  3. Change the Keyword for the search engins you like to use:

    • g: google.com
    • b: baidu.com
    • y: youtube.com
    • z: zhihu.com

  4. In Chrome address bar, g+(space/tab) will be google search, y+(space/tab) will be youtube search

Add Google Calendar Back to chrome://apps Page

  1. Navigate to https://calendar.google.com/calendar/r

  2. Press F12 to bring up the Developer Tools Panel.

  3. Click the Console tab.
