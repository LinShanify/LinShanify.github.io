The Ubuntu-Windows DualBoot will set the Ubuntu as the default operating system to boot. We can change the default boot to Windows by modifying the property GRUB_DEFAULT=4 in the ubuntu default grub file. (the number may vary)

Fix Chrome Opens Black Window for URL

google-chrome.desktop is missing the %U argument

  1. Open file:$HOME/.local/share/applications/google-chrome.desktop
  2. Find the line:Exec=/opt/google/chrome/chrome
  3. Add a space and %U: Exec=/opt/google/chrome/chrome %U
