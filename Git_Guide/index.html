Git Cache Github Credential in Linux/Mac

Turn on the credential helper so that Git will save your password in memory for some time.

Git Cache Github Credential in Windows

In Windows system, you need The Git Credential Manager for Windows (GCM) for Caching Github Credential. The GCM provides secure Git credential storage for Windows.
