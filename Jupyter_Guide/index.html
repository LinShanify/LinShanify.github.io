Jupyter Tips and Tricks

Set Up Password: Link

Magic Commands:

  • %matplotlib inline: inline graph plot
  • %load: insert the code from an external script %load ./hello_world.py
  • %%writefile:save the contents of that cell to an external file %%writefile hello_world.py
  • !:Executing Shell Commands !pip install numpy
  • %%time: Time information about a single run of the code in your cell.

Jupyter Widgets:

  • with pip

  • with conda
    conda install -c conda-forge ipywidgets

  • Additional For Jupyter Lab:

    1. install node.js
    2. run jupyter labextension install @jupyter-widgets/jupyterlab-manager

Jupyter Extension Manager:

A JupyterLab extension to facilitate the discovery and installation of other extensions.

  • Go into advanced settings editor.
  • Open the Extension Manager section.
  • Add the entry “enabled”: true.
  • Save the settings.
