Magic命令:

在Jupyter中可以使用许多魔法命令

  • %matplotlib inline: 设置 matplotlib，在jupyter中显示矢量图形
  • %load: 读取外部文件代码 %load ./hello_world.py
  • %%writefile:把这个cell的代码写入外部文件 %%writefile hello_world.py
  • !:在juyter cell里运行命令行程序 !pip install numpy
  • %%time: 这个cell的运行时间

Jupyter插件:

  • 用pip安装

  • 用conda安装
    conda install -c conda-forge ipywidgets

  • 如果使用Jupyter Lab的话需要激活:

    1. 安装node.js
    2. 在命令行运行 jupyter labextension install @jupyter-widgets/jupyterlab-manager

Jupyter 插件管理器:

  • 点击 advanced settings editor.
  • 选取 Extension Manager.
  • 添加 “enabled”: true.
  • 保存设定.
