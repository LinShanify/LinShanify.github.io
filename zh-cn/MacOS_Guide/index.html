MacOS鼠标滚轮反向: Scroll Reverser

MacOS的触板是自然滑动方向, 但是鼠标上还是习惯原来的滚动方式. Scroll Reverser可以反转鼠标的滚动方向但不改变触板的滚动方向

MacOS QuickLook 插件浏览Google WebP格式图片文件

WebP 是 Google的新照片格式 and MacOS系统原生并不支持预览 .webp后缀的文件. 这个插件能能让预览WebP照片文件,并显示图标 github
