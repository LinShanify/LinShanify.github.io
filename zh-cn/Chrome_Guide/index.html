Chrome地址栏快速切换搜索引擎

  1. 进入Chrome设置: chrome://settings/

  2. 修改 管理搜索引擎

  3. 可以给每一个搜索引擎一个关键字:

    • g: google.com
    • b: baidu.com
    • y: youtube.com
    • z: zhihu.com

  4. 在Chrome地址栏, g+(space/tab) 就是Google搜索, _y+(space/tab)_就是Youtube搜索

在Chrome App页面创建Google Calendar链接

Chrome App页面已经没有官方的Google Calendar软件了. 这个方法能创建一个URL链接的APP在Chrome APP页面. 略作修改也能用于其他的网站链接

  1. 打开Google Calendar页面

  2. 按F12打开开发者工具面板.

  3. 点击Console.

  4. 在Google Images里找一个Google Calendar的图片作为App的图标复制URL链接. 我用了这个图像作为例子:
    (https://png.icons8.com/color/1600/google-calendar.png)[https://png.icons8.com/color/1600/google-calendar.png]

  5. 在Console里复制运行下面的代码. 其他的URL的程序可以替换”link.href”更改其他的App图标样式:

    你会看到网站的图标变了

  6. 点击在Chrome右上角三个点,点击 More tools.. > Create shortcut.. 给快捷方式命名”Google Calendar”.

  7. 这时Chrome App页面就出现了Google Calendar了. 你也可以删除刚刚生成的快捷方式文件, 不影响Chrome App的页面.
