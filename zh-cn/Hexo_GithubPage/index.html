Hexo框架在 Github Pages上搭建个人网站

教程Hexo博客框架部署到Github Pages

创建个人Github Pages

创建一个新仓库(repository). 需要使用使用GitHub用户名: username.github.io, username是用户名.

Hexo框架

  1. 安装 Node.js
  2. 安装 Git (Windows版本)
  3. 安装Hexo框架 运行 npm install -g hexo-cli
  4. 创建一个本地网站博客文件夹
    • 创建一个文件夹
    • 在文件夹里, 运行hexo init初始化Hexo博客框架.
    • 运行npm install 安装所有的依赖库.
    • hexo g命令用来生成静态页面
    • hexo s命令用来创建一个临时本地服务器在4000端口 http://localhost:4000/
  5. 创建第一篇文章: hexo new 'My First Post'

部署Hexo博客到 Github Pages

  1. 安装hexo-deployer-git npm install hexo-deployer-git --save
  2. 修改_config.yml文件的deploy 属性
  3. 运行hexo d命令把Hexo博客推送到Github Pages的仓库里repository:

修改主题

这个网站使用的主题是minos. 因为自带多语言切换

安装主题:

在hexo/_config.yml文件里修改 theme: theme: minos.
