Fix EVP_CIPHER_CTX_cleanup error

when openssl version is 1.1.0 and above, starting shadowsocks2.8.2 server will show undefined symbol: EVP_CIPHER_CTX_cleanup error.

because in openssl1.1.0，they abandon EVP_CIPHER_CTX_cleanup function and change to EVP_CIPHER_CTX_reset.

Solution:

  1. edit ShaodowSocks openssl.py file
  2. go to line 52（shadowsocks2.8.2 version，other version search cleanup）
  3. go to line 111 (shadowsocks2.8.2 version，other version search cleanup）
